@camelsincaves When the query is statistical, it equals the estimand . 
Example: query Q=E[Y|x]=?    E_s=E[Y|x]      i.e. Same. 
Causal query Q=E{Y|do(x))=?  E_s  is not Q, it is  SUM_z E[Y||x,z]P(z) iff backdoor holds in the model.
