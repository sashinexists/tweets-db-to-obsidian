2/2 This equation reminded me of the rule for the conditional P(A|B) when B is partition (i.e. B1 + B2 + ... + Bn  = B), namely:

P(A|B) = P(A|B1)*P(B1|B) + P(A|B2)*P(B2|B) + ... + P(A|Bn)*P(Bn|B).
