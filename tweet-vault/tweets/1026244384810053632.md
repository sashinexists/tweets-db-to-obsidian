There are 3 basic steps to the structural causal model (SCM) approach:
1️⃣ abduction: use the data to estimate the U’s
2️⃣ action: change education to the value we want to do
3️⃣ prediction: calculate the potential outcome by combining 1 &amp; 2. https://t.co/ToijSnbY7k
