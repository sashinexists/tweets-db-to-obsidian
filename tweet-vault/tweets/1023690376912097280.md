The do(x) operator simulates what will happen if you actually apply the atomic action do(X=x). If you cant implement such action (eg. do(rain)) it still tells you what will happen if you could, and sometimes it does not matter, (eg, the grass will still get wet). #bookofwhy https://t.co/nEkstoAyC4
