@DToshkov "Counterfactual" is short for "contrary to facts." If we take this definition as a guideline, there is nothing counterfactual in RCT. Inferring Joe's behavior from Jane's is not contrary to fact, because there is no logical contradiction between Joe's and Jane's treatments.
