When answering questions of attribution, we should focus on this causal parameter: P(Y(A=0)=0|Y=1, A=1, X=x), where Y is a binary outcome, A is a binary exposure, Y(A=0) is a potl. outcome when A=0, X is covariates. Pearl calls it prob. of necessity, others prob. of causation. 4/
