✔Causal effect we want to estimate:
ℹPr(Y|do(X))
ℹProbability of Y if you set X to a value
❓What we get statistically from data:
🔘Pr(Y|X)

🔆Confounding is when
Pr(Y|do(X)) ≠ Pr(Y|X)
