@hazqiyal The bi-directed arrows denote correlated errors for some reasons, e.g., latent variables with names, un-named latents, selection bias, etc. If you suspect correlation, use them. Absence of those bi-directed arrows implies error-un-correlatedness for any reason. #Bookofwhy
